Rent listings bot

Мониторинг Avito/ЦИАН/Яндекс → уведомления в Telegram

Python 3.11+Telegram Bot APISQLiteDocker

Бот опрашивает выдачи, дедуплицирует объявления, отправляет сообщения в разные каналы (жилая/коммерция), умеет работать через прокси и поддерживает backoff при ограничениях площадок.

Показать детали
  • Фишки: раздельные пулы прокси по источникам, флаги сегментов, диагностические скрипты.
  • Результат: автоматизация мониторинга аренды и оперативные уведомления.
GitHub профиль репозиторий: локально `rent-listings-bot/`

Traffic landing demo

Статический лендинг под рекламный трафик

HTMLCSSJSЯндекс.МетрикаGitHub Pages

Шаблон “оффер → доверие → заявка”: mobile-first, липкий CTA на мобильных, заготовка под цели Метрики и события. Можно быстро клонировать и запускать гипотезы.

Показать детали
  • Фишки: якорная навигация, UX формы, цели `reachGoal`.
  • Результат: быстрый запуск лендинга и аналитики под трафик.

Warehouse demo (SPA + Electron)

Учёт склада, услуг и отчёты с экспортом

React 19TypeScriptViteTailwindElectron

Веб-приложение: склад (приход/расход, минимальные остатки), заказы со списанием, услуги по сотрудникам, отчёты с периодами и экспорт CSV для Excel. Данные — в localStorage (без бэкенда).

Показать детали
  • Фишки: KPI дашборд, графики, Ctrl+K поиск, undo/redo операций, тёмная/светлая тема.
  • Результат: демо, которое можно показать заказчику как “готовый продукт”.
GitHub профиль проект: локально `warehouse-demo/`

PythonTelegramWebhooksGoogle Sheets (mock)
